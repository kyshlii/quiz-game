Quick Reviewer

  • IP Spoofing - fake source IP used to hide identity or impersonate another host
  • MAC Spoofing - attacker copies another device MAC address
  • ICMP Recon - ping and ICMP used to discover active hosts
  • Smurf Attack - ICMP amplification using broadcast and spoofed victim IP
  • TCP SYN Flood - many SYNs create half-open connections
  • TCP Reset - RST packet forcefully ends a TCP session
  • Session Hijacking - attacker predicts sequence numbers and takes over session
  • UDP Flood - massive UDP traffic consumes bandwidth and resources
  • TCP - reliable, flow control, three-way handshake
  • UDP - connectionless, low overhead, no built-in reliability
